// 08 RECOMMENDED ACTIONS

Recommended Actions

ImmediateWeek 1-2
01

Add FAQPage Schema to Top 10 Specialties

Start with highest-traffic specialties (General Practice, Cardiology, Dermatology, Pediatrics, etc.). Create 5-10 common questions per specialty page with structured FAQPage schema. Example: 'When should I see a cardiologist?', 'What does a cardiology consultation involve?', 'How to prepare for a heart health checkup'.

02

Create 'How to Choose a Doctor in Hong Kong' Guide

Comprehensive guide targeting 'how to find a good doctor hong kong', 'how to choose a specialist', etc. with Q&A format. This fills a major content gap—no competitors have this. Format as extractable FAQ with schema markup.

03

Enable AI Bot Crawling Explicitly

Add explicit allow rules for GPTBot, ClaudeBot, PerplexityBot in robots.txt. While not currently blocked, explicit permission signals openness to AI crawlers.

Short-termMonth 1-2
04

Implement Physician Schema on Doctor Profiles

Add Physician schema to all 6,000+ doctor profiles with: name, specialty, qualifications, hospital affiliations, accepted insurance, languages spoken. This healthcare-specific markup signals authority to AI models.

05

Add MedicalBusiness Schema to Clinic Pages

Tag clinic/practice pages with MedicalBusiness schema including: name, address, phone, opening hours, services offered, accepted insurance.

06

Create Video Transcripts for FindDocTV

FindDoc has 400+ healthcare videos but no confirmed transcripts. Add full text transcripts to make video content machine-readable and citable by LLMs.

07

Build Comparison Pages: FindDoc vs Competitors

Create dedicated comparison pages: 'FindDoc vs DrGo', 'FindDoc vs DoctorNow', 'FindDoc vs TabMed' with comparison tables, use cases, and data-backed differentiators (6,000 doctors, 4.3M visits/year, founded 2012). Target ChatGPT's citation gaps.

Medium-termMonth 3-6
08

Launch 'Best Doctors in Hong Kong' Editorial Series

Create authoritative guides: 'Best Cardiologists in Hong Kong 2026', 'Top Pediatricians for Expat Families', 'Most Experienced Dermatologists in Central'. Use data from FindDoc's 6,000-doctor database. Format as extractable lists with criteria explanations.

09

Add Article Schema to Health Academy Content

Tag all blog posts, health guides, and educational content with Article or BlogPosting schema including author (doctor credentials), publish date, medical review date.

10

Implement Server-Side Rendering (SSR)

Current likely CSR setup means AI crawlers may see empty shells. Migrate to SSR or static site generation to ensure full content accessibility for AI bots.

11

Create llms.txt File

Add llms.txt to root directory with structured information about FindDoc: what it is, key features, data sources, citation guidelines. This emerging standard helps LLMs understand and cite platforms correctly.
